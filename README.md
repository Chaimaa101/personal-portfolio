## 🚀 Features

- **Interactive UI** with smooth animations using **Framer Motion**  
- **Dark/Light Mode** toggle for better accessibility and user experience  
- Fully **responsive design** for desktop, tablet, and mobile  
- Built with **React** for fast and dynamic front-end development  
- Styled using **Tailwind CSS** for a modern and clean design  

## 💻 Technologies Used

- **React.js** – Frontend library for building UI  
- **Tailwind CSS** – Utility-first CSS framework for styling  
- **Framer Motion** – Animation library for smooth transitions  
- **JavaScript (ES6+)** – Programming logic

This portfolio includes a **dark/light mode toggle**, enhancing user experience and accessibility.
